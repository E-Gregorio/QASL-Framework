Descripción
Como visitante del sitio de e-commerce
Quiero registrar una cuenta nueva con mi email, nombre y contraseña
Para poder acceder a las funcionalidades de compra del sitio.
Usuarios / Roles Visitante anónimo del sitio (sin sesión iniciada).
Reglas de Negocio
BR1: El email del usuario debe ser único en el sistema. No se permiten registros duplicados con el mismo email.
BR2: El password debe tener un mínimo de 6 caracteres alfanuméricos.
BR3: Los campos email, nombre y password son obligatorios para completar el registro.
BR4: Tras un registro exitoso debe mostrarse un mensaje de confirmación visible para el usuario.
