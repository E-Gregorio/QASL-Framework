## QASL Framework

Este framework representa la implementación práctica de metodologías de testing avanzadas, integrando:

- **Unit Testing** - Vitest (128+ tests del framework)
- **E2E Testing** - Playwright con Allure Reports
- **API Testing** - Newman/Postman Collections
- **Performance Testing** - K6 con métricas en Grafana
- **Security Testing** - OWASP ZAP Automation
- **AI-Powered Analysis** - Análisis SIGMA con Claude API
